# Queries

## Get a snapshot of which Entra users are logged in to hosts

**Compatibility: Windows**

```sql
SELECT
	upn,
	user,
	logon_domain,
	dns_domain_name,
	authentication_package
FROM
	logon_sessions
WHERE
	upn LIKE "< your email domain here >"
GROUP BY
	upn
```

## Analyse specific processes running to understand how much RAM/CPU they are consuming

**Compatibility: Mac, Windows, Linux**

In this example, we are looking at processes where the name contains `RTProtection` (short for "real-time protection", aka Antivirus/EDR-type softwares). We get their `pid` (process Id), `name` , `system_time` (which we divide by 86400 to convert to days), `start_time` (which we convert to dd/mm/yyyy format) and RAM consumed (`total_size` which we divide by 1048576 to have it in megabytes).

```sql
SELECT
	pid,
	name,
	ROUND((system_time / 86400), 2) AS system_time_days,
	strftime('%d/%m/%Y', datetime(start_time, 'unixepoch')) AS start_time_date,
	ROUND((total_size / 1048576), 2) AS total_size_mb
FROM
	processes
WHERE
	name LIKE '%RTProtection%';
```

## Get current IP from hosts

**Compatibility: Mac, Windows, Linux**

```sql
SELECT
	JSON_EXTRACT(result, '$.ip') AS ip,
	JSON_EXTRACT(result, '$.city') AS city,
	JSON_EXTRACT(result, '$.region') AS region,
	JSON_EXTRACT(result, '$.country') AS country
FROM
	curl
WHERE
	url = 'http://ipapi.co/json';
```

# Policies

## Check if device is still enrolled in Intune

**Compatibility: Windows**

This policy will fail if the device is still enrolled in Intune.

```sql
SELECT 1
FROM
	registry
WHERE
	path LIKE 'HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Enrollments\%%'
	AND name =  "DiscoveryServiceFullURL"
	AND data NOT IN (
"https://enrollment.manage.microsoft.com/enrollmentserver/discovery.svc",
"https://discovery.dm.microsoft.com/EnrollmentConfiguration?api-version=1.0"
)
```

## Check presence of applications

(for example to install an application if it is missing)

### Mac

```sql
SELECT 1 FROM apps WHERE bundle_identifier = 'your_bundle_identifier';
```

<Note>
To find the bundle identifier of an application, run the following command in Terminal: `osascript -e 'id of app "application_name"'`
</Note>

### Windows

```sql
SELECT 1 FROM programs WHERE name LIKE '%application_name%';
```

## Detect if an app is running

**Compatibility: Mac, Windows, Linux**

```sql
SELECT
	1
FROM
	processes
WHERE
	name LIKE '%process_name%';
```

## List all installed applications

### Mac

**Compatibility: Mac**

Returns all apps installed in `/Applications` and their bundle identifiers.

```sql
SELECT
	name,
	bundle_identifier,
	bundle_short_version AS version,
	last_opened_time
FROM
	apps
ORDER BY
	name;
```

### Windows

**Compatibility: Windows**

```sql
SELECT
	name,
	version,
	install_date,
	publisher
FROM
	programs
ORDER BY
	name;
```

## Get disk encryption status

**Compatibility: Mac, Windows**

### Mac (FileVault)

```sql
SELECT
	name,
	uuid,
	encrypted,
	type
FROM
	disk_encryption;
```

### Windows (BitLocker)

```sql
SELECT
	device_id,
	drive_letter,
	persistent_volume_id,
	conversion_status,
	protection_status
FROM
	bitlocker_info;
```

## Get OS version across your fleet

**Compatibility: Mac, Windows, Linux**

```sql
SELECT
	hostname,
	os_type,
	name AS os_name,
	version,
	build,
	platform
FROM
	os_version
JOIN
	system_info;
```

## List users with admin privileges

### Mac

**Compatibility: Mac**

Returns local users who are members of the `admin` group.

```sql
SELECT
	u.username,
	u.uid,
	u.directory
FROM
	users u
JOIN
	user_groups ug ON u.uid = ug.uid
JOIN
	groups g ON ug.gid = g.gid
WHERE
	g.groupname = 'admin';
```

### Windows

**Compatibility: Windows**

```sql
SELECT
	name,
	comment
FROM
	users
WHERE
	type = 'local'
	AND is_hidden = '0';
```

## Check last reboot time

**Compatibility: Mac, Windows, Linux**

```sql
SELECT
	hostname,
	datetime(last_reboot, 'unixepoch') AS last_reboot_date,
	days_since_update
FROM
	uptime
JOIN
	system_info;
```

## Find recently installed software (last 30 days)

### Mac

**Compatibility: Mac**

```sql
SELECT
	name,
	bundle_identifier,
	datetime(last_opened_time, 'unixepoch') AS last_opened
FROM
	apps
WHERE
	last_opened_time > (strftime('%s', 'now') - 2592000)
ORDER BY
	last_opened_time DESC;
```

### Windows

**Compatibility: Windows**

```sql
SELECT
	name,
	version,
	install_date,
	publisher
FROM
	programs
WHERE
	install_date > strftime('%Y%m%d', date('now', '-30 days'))
ORDER BY
	install_date DESC;
```

## Check SentinelOne agent status

**Compatibility: Mac**

Returns whether the SentinelOne agent is installed and its version.

```sql
SELECT
	name,
	bundle_identifier,
	bundle_short_version AS version
FROM
	apps
WHERE
	bundle_identifier LIKE 'com.sentinelone%';
```

**Compatibility: Windows**

```sql
SELECT
	name,
	version,
	publisher
FROM
	programs
WHERE
	name LIKE '%SentinelOne%'
	OR name LIKE '%Sentinel Agent%';
```
