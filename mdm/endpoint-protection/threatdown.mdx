<Note>
  **Deployment type: Automatic**

  ThreatDown is deployed **automatically** via MDM. Once you purchase ThreatDown through Primo and create an MDM control, the agent is pushed to all targeted devices without any manual steps. No user interaction is required.
</Note>

This article explains how to deploy ThreatDown on all or part of your fleet of devices.

## Prerequisites

- Have Primo MDM
- Have a ThreatDown license purchased through Primo

<Note>
  Automatic ThreatDown deployment is only available for licenses purchased through Primo. Contact us to get started.
</Note>

## Deploy ThreatDown to your fleet

Deploying ThreatDown on your fleet's devices is done through your MDM configurations.

1. Go to **MDM \> Controls \> Add new control**
2. Choose the platform (available platforms)
3. Enable the **Endpoint Detection and Response (EDR) - ThreatDown** card
4. Update the targeting to target the devices you want
5. Confirm

The ThreatDown agent will be automatically installed on all devices included in the control.

## Check available platforms

ThreatDown is currently available for the following platforms:

- **macOS** (arm64) - M1 chips and more
- **Windows** (x64) - Supports Windows Pro

If the platform is not mentioned above, you can still manually install ThreatDown by downloading the installer from the ThreatDown console and installing it manually with Fleet.
