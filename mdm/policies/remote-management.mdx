---

With Primo, you can turn on remote access for your devices through **RustDesk**. This is managed directly via an MDM control, so you don't need to install or configure anything manually. Once enabled, RustDesk is installed automatically on all devices targeted by the profile.

From the device drawer, you can then start a remote session right away. There's no need for users to share codes or passwords.

Sessions follow the same access rules you already set in Primo and are saved in the MDM audit log, so you always know who connected and when.

# Enable RustDesk in an MDM profile

1. Go to the **Profiles** section in the Primo cockpit.
2. Select a profile (or create a new one).
3. In the list of available controls, switch on **RustDesk**.

The RustDesk agent will then install automatically on all devices using this profile.

# Start a remote management session

<Warning>
  [**macOS**] On first run of the application, user will be asked to **accept permissions** such as Screen Sharing. These permissions cannot be deployed by a MDM and must be **approved** by the user.
</Warning>

1. Select the device you want to connect to and open its drawer.
2. Scroll to the "Remote Access" card and click **Start session**.
   - You can run the session in your browser or through the local app on your computer.

It may take up to 30 seconds for the session to be ready. The user does not need to do anything on their side.

# Security and access

- **No extra passwords**: Users never have to send codes or login details.
- **Logs available**: Every session is recorded so you can check past activity.
- **Encrypted connection**: The session is secure from start to finish.

This way you can connect to devices quickly while keeping access safe and controlled.
