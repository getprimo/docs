title: "Remote Access with RustDesk"
description: "Enable remote access for your devices effortlessly with RustDesk via Primo's MDM control, allowing instant remote sessions without the need for user passwords or manual configuration, while ensuring secure and logged connections."
---

Give your IT team one-click remote access to managed devices via RustDesk, deployed automatically through an MDM profile. No manual installation, no password sharing — sessions are encrypted, logged in the MDM audit trail, and available directly from the device drawer in Primo.

## OS support

| macOS | Windows | Linux | iOS / iPadOS | Android |
|-------|---------|-------|--------------|---------|
| ✅ | ✅ | ✅ | | |

## Enable RustDesk in an MDM profile

1. Go to the **Profiles** section in the Primo cockpit.
2. Select a profile (or create a new one).
3. In the list of available controls, switch on **RustDesk**.

The RustDesk agent will then install automatically on all devices using this profile.

## Start a remote management session

1. Select the device you want to connect to and open its drawer.
2. Scroll to the "Remote Access" card and click **Start session**.
   - You can run the session in your browser or through the local app on your computer.

It may take up to 30 seconds for the session to be ready. The user does not need to do anything on their side.

## On macOS

<Warning>
  On first run of the application, the user will be asked to **accept permissions** such as Screen Sharing. These permissions cannot be deployed by MDM and must be **approved** by the user.
</Warning>

After the user has granted the required permissions, subsequent sessions start without any user interaction.

## On Windows

RustDesk installs silently and requires no user interaction. Remote sessions can be started directly from the device drawer without any setup on the device side.

## On Linux

RustDesk installs automatically on Linux devices enrolled in the profile. Remote sessions work the same way as on other platforms — start the session from the device drawer in Primo.

## Security and access

- **No extra passwords**: Users never have to send codes or login details.
- **Logs available**: Every session is recorded so you can check past activity.
- **Encrypted connection**: The session is secure from start to finish.
