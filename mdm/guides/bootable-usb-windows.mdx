title: "Create a bootable USB drive and install Windows 11"

---

This article explains how to prepare a bootable USB drive with Windows 11, then use this drive to install Windows 11 on a computer.

# Prepare the bootable USB drive

1. **Download Microsoft's official tool**
   - Go to [Download Windows 11](https://www.microsoft.com/software-download/windows11).
   - Select *Create Windows 11 installation media*.
   - Download and run the *Media Creation Tool*.

2. **Insert and configure the USB drive**
   - Plug in a USB drive with at least **8 GB** capacity.
   - In the tool, select *USB flash drive* as the destination media.
   - Follow the steps to create the bootable USB drive.

   **Warning:** All data on the USB drive will be erased.

# Boot the PC from the USB drive

1. **Plug the USB drive into the target PC**.
2. **Enter the boot menu**
   - Turn on the PC and open the *Boot Menu* or *BIOS/UEFI* (key depends on manufacturer, often F2, F12, ESC, or DEL).
3. **Select the USB drive as boot device**
   - Save and restart the computer.
   - Windows 11 installation will start from the USB drive.

# Install Windows 11

1. **Follow the installation wizard**
   - Select language, time, and keyboard settings.
   - Click *Install now*.
2. **Enter the product key (if required)**.
3. **Choose the installation type**
   - *Upgrade* to keep files and applications (if a compatible version of Windows is installed).
   - *Custom* for a clean installation.
4. **Select the destination drive**
   - Delete, format, or create partitions if needed.
   - Start the installation.

The computer will restart several times before completing Windows 11 setup.
