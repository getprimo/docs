title: "Create and manage shipments"
description: "Learn how to efficiently create and manage shipments in Primo, whether during the offboarding process or manually, by following the outlined steps for configuring shipment details and accessing shipping labels."
---

This article explains how to create a shipment in Primo and configure the associated shipping label. You can create a shipment during the offboarding flow or directly from the Primo Cockpit. The configuration steps are the same in both cases.

There are two entry points:

- During the offboarding flow (recommended when offboarding an employee)
- From Orders & Shipments → Shipments → New (manual creation)

## Create a shipment

### From offboarding flow

Use this method when triggering an offboarding. The employee's address is automatically pre-filled.

1. Go to the employee's **Offboarding** page.
2. Navigate to the **MANAGE EQUIPMENT** step.
3. Enable the toggle **Create a shipping label**.

You are redirected to the shipment creation page.

From this point, follow the standard shipment configuration steps described below.

### From the Primo Cockpit

Use this method to organise a shipment without going through the offboarding flow.

1. Go to **Orders & Shipments → Shipments → New**.
2. Select the device and/or accessories to ship.
3. Click **Next**.

You can then configure the shipment as described below.

## Configure the shipment details

The following steps apply to both entry points.

1. Select or confirm the **sender address**.
2. Select the **recipient address**.
3. In **SHIPMENT NOTIFICATIONS**, add the e-mail address(es) that should receive the shipping label (for example: the employee, an HR admin, or a shared inbox).
4. Select the **billing profile** to define who will be billed for the shipment.
5. Optional: add a **return box** purchase. The box will be sent to the sender so they can pack the device and return it using the provided label.
6. Click **Validate** to confirm and generate the shipping label.

## Access the shipping label

- The shipping label is sent by e-mail within 24 hours to the address(es) entered in **SHIPMENT NOTIFICATIONS**.
- The label is also available in the Primo Cockpit under **Orders & Shipments → Shipments → Label**.

## Return process for the employee

### Receive the shipping label

The employee (or any admin specified in **SHIPMENT NOTIFICATIONS**) receives the shipping label by e-mail within 24 hours.

### Package the device

- If a return box was purchased: The box is sent to the employee. They place the device inside, apply the shipping label, and hand the package to the carrier.
- If no box was purchased: The employee must package the device securely and apply the received shipping label.

### Track the shipment

- Tracking information is available via the shipping e-mail.
- Tracking is also available in the Primo Cockpit under **Orders & Shipments → Shipments**.
