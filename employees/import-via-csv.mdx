This article explains how to manage your employee directory manually when no HRIS is connected.

## Use Cases

Use this method if:

- You don't use an HRIS
- You're doing a one-time import
- You want to add external collaborators (freelancers, interns, etc.)

Note that by importing your employees, it will be necessary to manually enter the arrivals and departures of your employees in Primo.

## Manual employee synch process

Here is the information you will be able to import into Primo and what your CSV file must contain to correctly import employees:

- First name *
- Last name *
- Professional email *
- Personal email (optional)
- Phone number (optional)
- Team (optional)
- Job title (optional)

You will find a template at the bottom of this page that you can download and complete to help you import your employees.

To import employees manually:

1. Go to **Settings > Employee Synch**
2. Click on "Upload CSV or Employees"
3. Select the file you want to import
4. Match Primo's titles with those in your file
5. Verify the information and import
6. Click **Save**. The employee will appear in your employees list

## Manual updates

- You can **edit** any field at any time
- Fields are **never overwritten** unless you connect an HRIS later

<Warning>
**Limitations**

- No automated onboarding/offboarding flows
- No sync with contract dates
- Manual cleanup required if employees leave
</Warning>
