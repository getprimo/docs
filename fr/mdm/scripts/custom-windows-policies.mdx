Étendez vos profils Windows MDM avec des politiques CSP (Configuration Service Provider) personnalisées à l'aide de XML brut. Utilisez-le lorsque les contrôles intégrés de Primo ne couvrent pas un paramètre Windows spécifique que vous devez appliquer, tel que la personnalisation du bureau, le mode kiosque ou d'autres configurations d'entreprise.

## Prise en charge du système d'exploitation

| macOS | Windows | Linux | iOS / iPadOS | Android |
|-------|---------|-------|--------------|---------|
| | ✅ | | | |

## Sur Windows

<Note>
Lors de l'écriture d'un XML pour appliquer une stratégie CSP à un périphérique, omettez les premières (et dernières) lignes du fichier XML et commencez (et terminez) par `<Replace>@@I18N_SEG_10@@<Add>@@I18N_SEG_11@@</Replace>@@I18N_SEG_12@@</Add>`)
</Note>

### Autoriser le CSP de personnalisation sur Windows Pro

```xml
<Replace>
  <!-- Allows Personalization CSPs to be deployed on non-Enterprise versions of Windows.
       Setting this value to true triggers the action to configure a device as education environment.
       Thorough testing is crucial to ensure it does not disrupt workflows or create compatibility issues. -->
  <Item>
    <Meta>
      <Format xmlns="syncml:metinf">bool</Format>
    </Meta>
    <Target>
      <LocURI>./Vendor/MSFT/SharedPC/SetEduPolicies</LocURI>
    </Target>
    <Data>true</Data>
  </Item>
</Replace>
```

### Définir l'image d'arrière-plan et l'écran de verrouillage

<Note>
Le CSP de personnalisation est pris en charge dans les éditions Windows Enterprise et Education. Il fonctionne dans Windows Professional uniquement lorsque SetEduPolicies dans SharedPC CSP est défini. Voir « Autoriser le CSP de personnalisation sur Windows Pro ».
</Note>

```xml
<Replace>
  <CmdID>1</CmdID>
  <Item>
    <Target>
      <LocURI>
        ./Vendor/MSFT/Personalization/LockScreenImageUrl
      </LocURI>
    </Target>
    <Meta>
      <Format xmlns="syncml:metinf">chr</Format>
      <Type>text/plain</Type>
    </Meta>
    <Data> … your URL here … </Data>
  </Item>
</Replace>
<Replace>
  <CmdID>2</CmdID>
  <Item>
    <Target>
      <LocURI>
        ./Vendor/MSFT/Personalization/DesktopImageUrl
      </LocURI>
    </Target>
    <Meta>
      <Format xmlns="syncml:metinf">chr</Format>
      <Type>text/plain</Type>
    </Meta>
    <Data> … your URL here … </Data>
  </Item>
</Replace>
```
