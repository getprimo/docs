### Remarques

1. **ChromeOS** : bien que vous puissiez inscrire des appareils ChromeOS, le contrôle de ces appareils (à la fois en termes de déploiement d'applications et de paramètres du système d'exploitation) s'effectue dans votre console d'administration Google. Pour plus de détails, contactez le support.
2. **Linux :** Les appareils Linux sont difficiles à chiffrer, car cela peut parfois nécessiter d'effacer le disque dur et de configurer un système d'exploitation à partir de zéro. Cependant, avec Primo, vous pouvez toujours centraliser les rapports sur les appareils cryptés ou non, obsolètes, etc.
3. Voici les **distributions Linux** que nous prenons en charge :
   - ✅Ubuntu 20.04+
   - ✅ Debian 11+
   - ✅ CentOS 7.1+
   - ✅ Fedora 38+
   - ✅ Amazon Linux 2+
   - ✅ Red Hat Enterprise Linux (RHEL) 7+
   - ✅ ouvertSUSE 15.6+
   - ✅ Arch Linux
   - ✅Omarchie
   - ❌ Alpin (format APK)
4. FleetDM préfère les versions x86 64 bits des applications lorsqu'elles sont disponibles. L'installation sur des hôtes Arm (par exemple dans une VM sur une machine Apple Silicon) peut ne pas fonctionner ou avoir d'autres conséquences inattendues.
