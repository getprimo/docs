L'installation du MDM est une étape cruciale dans la gestion de flotte. Cette procédure est simple et rapide : elle ne prend que quelques minutes et est obligatoire pour tous les utilisateurs.

## Options d'inscription

Primo prend en charge trois façons d'inscrire un appareil dans MDM :

### Option 1 — Invitation par courrier électronique

Envoyez un e-mail d'inscription personnalisé directement depuis le cockpit Primo. Chaque employé reçoit un lien unique pour installer l'agent MDM sur son appareil.

**Envoyer une invitation :**
- **Pour 1 employé :** ouvrez le profil de l'employé, puis cliquez sur **Inviter à MDM**.
- **Pour une sélection d'employés :** sélectionnez les employés souhaités, cliquez sur **Inviter à MDM**, puis sur **Inviter X employés sélectionnés**.
- **À tous les employés :** cliquez sur **Inviter à MDM**, puis sur **Inviter les employés correspondants**.

Une fois initiés, les employés reçoivent un e-mail de Primo (support@getprimo.com) avec pour objet « Sécurisez votre équipement avec Primo ». L'e-mail comprend :
- Un **lien d'inscription personnel** unique à cet employé.
- Un guide d'installation étape par étape.

<Note>
Le lien d'inscription est valable **7 jours**. Passé ce délai, vous devrez envoyer une nouvelle invitation.
</Note>

### Option 2 — URL d'inscription

Chaque collaborateur dispose d'une page d'inscription dédiée accessible via une URL personnelle. Vous pouvez partager cette URL directement (par exemple via Slack ou votre intranet) comme alternative au courrier électronique.

L'URL se trouve dans le profil du salarié dans le cockpit Primo.

### Option 3 — Forfaits téléchargeables

Pour les déploiements gérés ou les environnements dans lesquels la messagerie électronique n'est pas adaptée, Primo fournit des packages d'installation téléchargeables (macOS `.pkg`, Windows `.exe`) qui peuvent être distribués via vos outils de déploiement de logiciels existants (par exemple Jamf, SCCM ou installation manuelle).

Les packages peuvent être téléchargés à partir de la section d'inscription du cockpit Primo et déployés silencieusement sans nécessiter l'interaction des employés.
