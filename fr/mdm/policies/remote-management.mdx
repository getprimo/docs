Donnez à votre équipe informatique un accès à distance en un clic aux appareils gérés via RustDesk, déployés automatiquement via un profil MDM. Pas d'installation manuelle, pas de partage de mot de passe : les sessions sont cryptées, enregistrées dans la piste d'audit MDM et disponibles directement depuis le tiroir de l'appareil.

## Prise en charge du système d'exploitation

| macOS | Windows | Linux | iOS / iPadOS | Android |
|-------|---------|-------|--------------|---------|
| ✅ | ✅ | ✅ | | |

## Activer RustDesk dans un profil MDM

1. Accédez à la section **Profils** dans le cockpit Primo.
2. Sélectionnez un profil (ou créez-en un nouveau).
3. Dans la liste des contrôles disponibles, activez **RustDesk**.

L'agent RustDesk s'installera alors automatiquement sur tous les appareils utilisant ce profil.

## Démarrer une session de gestion à distance

1. Sélectionnez l'appareil auquel vous souhaitez vous connecter et ouvrez son tiroir.
2. Faites défiler jusqu'à la carte « Accès à distance » et cliquez sur **Démarrer la session**.
   - Vous pouvez exécuter la session dans votre navigateur ou via l'application locale sur votre ordinateur.

Cela peut prendre jusqu'à 30 secondes pour que la session soit prête. L’utilisateur n’a rien à faire de son côté.

## Sur macOS

<Warning>
  Lors de la première exécution de l'application, il sera demandé à l'utilisateur d'**accepter les autorisations** telles que le partage d'écran. Ces autorisations ne peuvent pas être déployées par MDM et doivent être **approuvées** par l'utilisateur.
</Warning>

Une fois que l'utilisateur a accordé les autorisations requises, les sessions suivantes démarrent sans aucune interaction de l'utilisateur.

## Sur Windows

RustDesk s'installe silencieusement et ne nécessite aucune interaction de l'utilisateur. Les sessions à distance peuvent être démarrées directement à partir du tiroir de l'appareil sans aucune configuration côté appareil.

## Sur Linux

RustDesk s'installe automatiquement sur les appareils Linux inscrits dans le profil. Les sessions à distance fonctionnent de la même manière que sur les autres plateformes : démarrez la session à partir du tiroir de l'appareil.

## Sécurité et accès

- **Pas de mots de passe supplémentaires** : les utilisateurs n'ont jamais besoin d'envoyer de codes ou d'informations de connexion.
- **Journaux disponibles** : chaque session est enregistrée afin que vous puissiez vérifier l'activité passée.
- **Connexion cryptée** : La session est sécurisée du début à la fin.
