## Recommandez les licences Windows Pro pour vos ordinateurs

Windows Home n’est pas considéré comme adapté à un usage professionnel par opposition à un usage personnel. Ses limites en termes de sécurité, de gestion et de fonctionnalités professionnelles le rendent moins adapté aux environnements professionnels.

Voici les principales limitations de Windows Home qui le rendent moins adapté aux besoins des entreprises :

1. **Manque de fonctionnalités de sécurité avancées**
   - Absence de BitLocker : Windows Home ne dispose pas de BitLocker, l'outil de chiffrement de disque qui protège les données en cas de perte ou de vol de l'appareil.
   - Limites de la protection contre les menaces : Windows Home ne dispose pas de certaines protections avancées contre les menaces disponibles dans Windows Pro (WIP et Windows Defender System Guard)

2. **Gestion et administration limitées**
   - Impact limité de MDM : Bien que Windows Home puisse être géré par certaines solutions MDM, les fonctionnalités disponibles sont moins étendues que celles proposées par Windows Pro.
   - Absence de gestion de domaine et d'ID Microsoft Entra : Windows Home ne permet pas de rejoindre un domaine de serveur Windows ou un ID Microsoft Entra (anciennement Azure Active Directory), limitant ainsi les options d'authentification et de gestion des identités.

3. **Support et compatibilité**
   - Compatibilité réduite avec les applications professionnelles : certaines applications d'entreprise et outils de productivité sont conçus et testés principalement pour Windows Pro, garantissant ainsi une meilleure compatibilité et de meilleures performances.

<Note>
Windows Pro offre des fonctionnalités avancées spécialement conçues pour répondre aux besoins des entreprises en termes de sécurité des données, de gestion des appareils, d'assistance au travail à distance et d'intégration avec d'autres services d'entreprise, justifiant ainsi l'investissement supplémentaire pour les entreprises.
</Note>

## Licences Windows Home et MDM

Les deux types de licences (Home et Pro) peuvent être inscrits dans MDM. Les licences Windows Home permettent toujours une gestion plus limitée de vos équipements et un déploiement partiel des politiques de sécurité.

Plus d'informations : [Plateformes et systèmes d'exploitation pris en charge](/fr/mdm/get-started/supported-platforms)
