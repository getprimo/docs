<Note>
Installer le MDM sur les appareils de vos collaborateurs permet de les créer automatiquement dans le cockpit sans aucune action manuelle ni importation de votre part. Une fois inscrit, l'appareil apparaît dans votre onglet Fleet et est attribué à l'employé.
</Note>

Si vous choisissez quand même d’importer votre matériel, voici quelques informations à garder à l’esprit.

## Conditions préalables

Pour importer un équipement dans Primo, seul un nom est requis. Cependant, pour avoir une bonne visibilité sur votre flotte, nous vous recommandons d'importer un maximum d'informations. Cela vous aide à conserver des enregistrements précis.

## Procédure d'importation

Pour importer votre équipement dans le cockpit Primo :

1. Connectez-vous à votre cockpit Primo
2. Accédez à l'onglet Mon Fleet
3. Utilisez le bouton "Ajouter"
4. Sélectionnez votre fichier CSV préparé
5. Vérifier le mappage des colonnes
6. Confirmer l'importation

## Après l'importation

Une fois l'importation terminée :

- Vérifiez que tous les équipements apparaissent correctement dans votre inventaire
- Confirmez que les affectations des employés sont correctes
- Complétez les informations manquantes si nécessaire

Les équipements importés peuvent être modifiés ou mis à jour ultérieurement depuis le cockpit Primo.
