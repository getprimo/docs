Cet article explique comment créer un envoi et configurer l'étiquette d'expédition associée. Vous pouvez créer un envoi pendant le flux d'offboarding ou directement depuis le cockpit. Les étapes de configuration sont les mêmes dans les deux cas.

Il existe deux points d'entrée :

- Pendant le flux de départ (recommandé lors du départ d'un employé)
- Depuis Commandes et expéditions → Expéditions → Nouveau (création manuelle)

## Créer un envoi

### Du flux de délocalisation

Utilisez cette méthode lors du déclenchement d’un offboarding. L'adresse du salarié est automatiquement pré-remplie.

1. Accédez à la page **Désintégration** de l'employé.
2. Accédez à l'étape **GÉRER L'ÉQUIPEMENT**.
3. Activez le bouton **Créer une étiquette d'expédition**.

Vous êtes redirigé vers la page de création d'envoi.

À partir de ce point, suivez les étapes de configuration d’expédition standard décrites ci-dessous.

### Depuis le cockpit du Primo

Utilisez cette méthode pour organiser un envoi sans passer par le flux de délocalisation.

1. Accédez à **Commandes et expéditions → Expéditions → Nouveau**.
2. Sélectionnez l'appareil et/ou les accessoires à expédier.
3. Cliquez sur **Suivant**.

Vous pouvez ensuite configurer l'envoi comme décrit ci-dessous.

## Configurer les détails de l'expédition

Les étapes suivantes s'appliquent aux deux points d'entrée.

1. Sélectionnez ou confirmez l'**adresse de l'expéditeur**.
2. Sélectionnez l'**adresse du destinataire**.
3. Dans **NOTIFICATIONS D'EXPÉDITION**, ajoutez la ou les adresses e-mail qui doivent recevoir l'étiquette d'expédition (par exemple : l'employé, un administrateur RH ou une boîte de réception partagée).
4. Sélectionnez le **profil de facturation** pour définir qui sera facturé pour l'envoi.
5. Facultatif : ajoutez un achat **boîte de retour**. La boîte sera envoyée à l'expéditeur afin qu'il puisse emballer l'appareil et le retourner en utilisant l'étiquette fournie.
6. Cliquez sur **Valider** pour confirmer et générer l'étiquette d'expédition.

## Accéder à l'étiquette d'expédition

- L'étiquette d'expédition est envoyée par e-mail dans les 24 heures à la ou aux adresses renseignées dans **NOTIFICATIONS D'EXPÉDITION**.
- L'étiquette est également disponible dans le cockpit Primo sous **Commandes et expéditions → Expéditions → Étiquette**.

## Processus de retour du salarié

### Recevez l'étiquette d'expédition

L'employé (ou tout administrateur spécifié dans **NOTIFICATIONS D'EXPÉDITION**) reçoit l'étiquette d'expédition par e-mail dans les 24 heures.

### Emballer l'appareil

- Si une boîte de retour a été achetée : La boîte est envoyée au collaborateur. Ils placent l’appareil à l’intérieur, apposent l’étiquette d’expédition et remettent le colis au transporteur.
- Si aucune boîte n'a été achetée : L'employé doit emballer l'appareil en toute sécurité et appliquer l'étiquette d'expédition reçue.

### Suivre l'expédition

- Les informations de suivi sont disponibles via l’e-mail d’expédition.
- Le suivi est également disponible dans le cockpit Primo sous **Commandes et expéditions → Expéditions**.
