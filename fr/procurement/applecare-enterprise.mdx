## Acheter les garanties AppleCare Enterprise (ACE)

Les garanties [AppleCare Enterprise](https://www.apple.com/fr/support/professional/enterprise/) sont l'offre professionnelle d'Apple. Pour acheter ces garanties, vous devez créer un compte dédié, qui vous permettra d'obtenir un **'ID d'inscription' pour vos futurs achats de garantie.**

## Créez un compte AppleCare Enterprise (si vous n'en avez pas déjà un)

<Note>
  Vous devez disposer d'un compte Apple Business.
</Note>

- Inscrivez-vous sur le site https://enterprise.apple.com/directEnroll
- Sélectionnez _ACE_
- Remplissez les informations demandées sur votre entreprise
- Remplissez les informations de votre contact acheteur
- Lire et accepter les termes et conditions
- Enregistrez votre identifiant d'inscription

Vous recevrez un e-mail d'Apple contenant votre numéro d'inscription, que vous pourrez fournir lors de votre commande.

## Lors de votre première commande avec Primo

- Lorsque vous passez **votre première commande avec Primo**, sélectionnez « Ajouter une garantie » et fournissez votre « ID d'inscription ».
- Pour les **commandes suivantes**, cet identifiant d'inscription sera pré-rempli et devra être revalidé.

Une fois votre première commande activée, Apple créera un MSA (Master Service Agreement) vous permettant de continuer à ajouter des produits au contrat.

## Comment ouvrir un dossier de support avec AppleCare Enterprise

1. **Accédez à la console** https://enterprise.apple.com/
2. **Accédez aux documents de votre entreprise sur la page Accord**
    ![Capture d'écran : Comment ouvrir un dossier d'assistance avec Apple Care Enterprise](/images/notion-import/buy-apple-care-enterprise-guarantees__img-111__block-0025__2af43d9f.png)

3. **Ouvrir un dossier**

   **Option 1 : Votre employé ouvre le dossier :** Vous pouvez partager le document de support utilisateur final ACE avec les employés ou ils peuvent contacter le bon numéro et fournir le code PIN de l'entreprise.

   **Option 2 : Vous pouvez l'ouvrir vous-même :** Créez un dossier directement dans la console AppleCare Enterprise.
