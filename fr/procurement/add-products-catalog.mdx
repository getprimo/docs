Primo gère la disponibilité des produits **par pays**. Avant de pouvoir passer des commandes pour un pays donné, ce pays doit être configuré dans les paramètres de votre espace de travail.

## Ajoutez un pays à votre catalogue

Chaque pays depuis lequel vous souhaitez commander ou expédier doit être ajouté individuellement :

1. Accédez à **Commandes et expéditions \> Paramètres**.
2. Cliquez sur **Ajouter un pays**.
3. Sélectionnez le pays dans la liste.
4. Configurez tous les paramètres spécifiques au pays (par exemple, opérateur préféré, paramètres fiscaux).
5. Sauvegarder.

Répétez l’opération pour chaque pays dans lequel vous avez des employés ou souhaitez vous procurer du matériel.

<Note>
  Les pays sont gérés indépendamment : l’ajout d’un pays n’affecte pas le catalogue ou les paramètres des autres pays.
</Note>

## Demander un produit manquant

Si un produit est disponible dans votre pays mais n'est pas encore répertorié dans le catalogue Primo :

1. Ouvrez la section **Catalogue** dans votre cockpit.
2. Cliquez sur **Ajouter** dans le coin supérieur droit.
3. Remplissez les détails du produit :
   - Nom complet du produit et numéro de modèle
   - Informations du fabricant
   - Spécifications techniques
   - Pays où le produit devrait être disponible
4. Soumettre.

Notre équipe examinera votre demande et ajoutera le produit au catalogue du pays approprié.
