Estendi i tuoi profili Windows MDM con criteri CSP (Configuration Service Provider) personalizzati utilizzando XML non elaborato. Utilizzalo quando i controlli integrati di Primo non coprono un'impostazione specifica di Windows che devi applicare, ad esempio la personalizzazione del desktop, la modalità tutto schermo o altre configurazioni aziendali.

## Su Windows

<Note>
Quando scrivi un XML per applicare una policy CSP a un dispositivo, ometti le prime (e ultime) righe del file XML e inizia (e termina) con `<Replace>` o `<Add>` (rispettivamente `</Replace>` e `</Add>`)
</Note>

### Consenti CSP personalizzazione su Windows Pro

```xml
<Replace>
  <!-- Allows Personalization CSPs to be deployed on non-Enterprise versions of Windows.
       Setting this value to true triggers the action to configure a device as education environment.
       Thorough testing is crucial to ensure it does not disrupt workflows or create compatibility issues. -->
  <Item>
    <Meta>
      <Format xmlns="syncml:metinf">bool</Format>
    </Meta>
    <Target>
      <LocURI>./Vendor/MSFT/SharedPC/SetEduPolicies</LocURI>
    </Target>
    <Data>true</Data>
  </Item>
</Replace>
```

### Imposta l'immagine di sfondo e la schermata di blocco

<Note>
Il CSP di personalizzazione è supportato nelle edizioni Windows Enterprise ed Education. Funziona in Windows Professional solo quando è impostato SetEduPolicies in SharedPC CSP. Vedere "Consentire la personalizzazione CSP su Windows Pro".
</Note>

```xml
<Replace>
  <CmdID>1</CmdID>
  <Item>
    <Target>
      <LocURI>
        ./Vendor/MSFT/Personalization/LockScreenImageUrl
      </LocURI>
    </Target>
    <Meta>
      <Format xmlns="syncml:metinf">chr</Format>
      <Type>text/plain</Type>
    </Meta>
    <Data> … your URL here … </Data>
  </Item>
</Replace>
<Replace>
  <CmdID>2</CmdID>
  <Item>
    <Target>
      <LocURI>
        ./Vendor/MSFT/Personalization/DesktopImageUrl
      </LocURI>
    </Target>
    <Meta>
      <Format xmlns="syncml:metinf">chr</Format>
      <Type>text/plain</Type>
    </Meta>
    <Data> … your URL here … </Data>
  </Item>
</Replace>
```
