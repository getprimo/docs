<Note>
L'installazione di MDM sui dispositivi dei tuoi dipendenti consente di crearli automaticamente nella cabina di pilotaggio senza alcuna azione manuale o importazione da parte tua. Una volta registrato, il dispositivo viene visualizzato nella scheda Fleet e viene assegnato al dipendente.
</Note>

Se scegli comunque di importare la tua attrezzatura, ecco alcune informazioni da tenere a mente.

## Prerequisiti

Per importare l'attrezzatura in Primo, è richiesto solo un nome. Tuttavia, per avere una buona visibilità della tua flotta, ti consigliamo di importare quante più informazioni possibili. Questo ti aiuta a mantenere registrazioni accurate.

## Importa la preparazione del file

Ecco un file modello che ti aiuterà a importare facilmente la tua attrezzatura:

| **Criteri** | **Formato richiesto** | **Campo obbligatorio?** |
|---|---|---|
| E-mail del dipendente (se dispositivo assegnato) | E-mail | No |
| Tipo di dispositivo | COMPUTER PORTATILE<br/>SMARTPHONE<br/>TABLET | Sì |
| Nome del dispositivo | | Sì |
| Sistema operativo | WINDOWS<br/>MACOS<br/>LINUX<br/>ANDROID<br/>IOS<br/>Sconosciuto | No |
| Numero di serie | | No |
| Dimensioni dello schermo | | No |
| Linguaggio della tastiera | FRANCESE | No |
| Marchio | | No |
| Anno | | No |
| Note | | No |

## Procedura di importazione

Per importare la tua attrezzatura nella cabina di pilotaggio Primo:

1. Accedi al tuo abitacolo Primo
2. Vai alla scheda Il mio Fleet
3. Utilizzare il pulsante "Aggiungi".
4. Seleziona il file CSV preparato
5. Verificare la mappatura delle colonne
6. Conferma l'importazione

## Dopo l'importazione

Una volta completata l'importazione:

- Verifica che tutta l'attrezzatura appaia correttamente nel tuo inventario
- Confermare che le assegnazioni dei dipendenti siano corrette
- Completa eventuali informazioni mancanti, se necessario

L'attrezzatura importata può essere modificata o aggiornata successivamente dalla cabina di pilotaggio Primo.
