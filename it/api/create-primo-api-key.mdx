Primo Le chiavi API vengono utilizzate per autenticare le richieste a [Primo REST API](/it/api-reference). Ogni chiave ha come ambito l'area di lavoro dell'azienda.

## Genera una chiave API

1. Vai a **Impostazioni** > **API**.
2. Fare clic su **Crea chiave API**.
3. Assegna un nome alla chiave (ad esempio "La mia integrazione").
4. Fare clic su **Crea**.
5. Copia immediatamente la chiave: non verrà più visualizzata.

<Warning>
Conserva la chiave API in modo sicuro. Chiunque abbia questa chiave può accedere al tuo spazio di lavoro Primo tramite API.
</Warning>

## Utilizzare la chiave API

Passa la chiave nell'intestazione `Authorization` delle tue richieste:

```bash
curl https://api.getprimo.com/v1/devices \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Revoca una chiave

Per revocare una chiave, vai su **Impostazioni** > **API**, trova la chiave e fai clic su **Elimina**.
