---
title: "Crea chiavi API per Fleet"
description: "Genera una chiave FleetDM API per autenticare le richieste dirette alla tua istanza Fleet."
---

Fleet (FleetDM) Le chiavi API ti consentono di interagire direttamente con l'istanza Fleet, ad esempio per eseguire script, interrogare i dati del dispositivo o integrarsi con altri strumenti.

È possibile creare chiavi Fleet API direttamente da Primo o tramite il dashboard Fleet.

## Genera una chiave Fleet API da Primo

1. Vai a **Impostazioni** > **Sviluppatori** > **Chiavi Fleet API**.
2. Fare clic su **Crea chiave API**.
3. Assegna un nome alla chiave e configura le autorizzazioni.
4. Fare clic su **Crea**.
5. Copia immediatamente il token: non verrà più mostrato.

<Warning>
Mantieni segreto il tuo token Fleet API e non condividerlo. Ha le autorizzazioni configurate durante la creazione.
</Warning>

## Genera una chiave Fleet API dal dashboard Fleet

1. Apri il dashboard **Fleet** (accessibile dal pannello di controllo Primo tramite **Impostazioni** > **MDM** > **Apri Fleet**).
2. Fai clic sul tuo avatar nell'angolo in alto a destra e vai su **Il mio account**.
3. Scorri fino a **Token API**.
4. Fai clic su **Ottieni token API**.
5. Copia il token: è legato al tuo account utente Fleet.

<Warning>
Il tuo token Fleet API ha le stesse autorizzazioni del tuo account utente Fleet. Mantienilo segreto e non condividerlo.
</Warning>

## Genera una chiave Fleet API dalla CLI

È inoltre possibile creare chiavi Fleet API a livello di programmazione utilizzando la CLI Fleet:

```bash
fleetctl login --email your-email@example.com --url https://<your-fleet-instance>
fleetctl get api-token
```

Oppure crea una chiave API direttamente tramite Fleet API:

```bash
curl -X POST https://<your-fleet-instance>/api/v1/fleet/users/<user-id>/api_token \
  -H "Authorization: Bearer YOUR_EXISTING_TOKEN" \
  -H "Content-Type: application/json"
```

## Utilizzare la chiave Fleet API

Passa il token nell'intestazione `Authorization`:

```bash
curl https://<your-fleet-instance>/api/v1/fleet/hosts \
  -H "Authorization: Bearer YOUR_FLEET_API_TOKEN"
```

## Rigenera o revoca un token

Fleet I token API non scadono automaticamente. Per revocare l'accesso, vai su **Il mio account** in Fleet e fai clic su **Rigenera token API**. Ciò invalida immediatamente il token precedente.
