<Note>
Installing the MDM on your employees' devices allows them to be automatically created in the cockpit without any manual action or import on your part! Once enrolled, the device appears in your Fleet tab and is assigned to the employee.
</Note>

If you still choose to import your equipment, here is some information to keep in mind

## Prerequisites

To import equipment into Primo, only a name is required. However, to have good visibility of your fleet, we recommend importing as much information as possible. This will help you keep better track of everything!

## Import file preparation

Here is a template file that will help you easily import your equipment:

| **Criteria** | **Required Format** | **Mandatory Field?** |
|---|---|---|
| Employee email (if device assigned) | Email | No |
| Device type | LAPTOP<br/>SMARTPHONE<br/>TABLET | Yes |
| Device name | | Yes |
| Operating system | WINDOWS<br/>MACOS<br/>LINUX<br/>ANDROID<br/>IOS<br/>UNKNOWN | No |
| Serial number | | No |
| Screen size | | No |
| Keyboard language | FRENCH | No |
| Brand | | No |
| Year | | No |
| Notes | | No |

## Import procedure

To import your equipment into the Primo cockpit:

1. Log in to your Primo cockpit
2. Go to the My Fleet tab
3. Use the "Add" button
4. Select your prepared CSV file
5. Verify the column mapping
6. Confirm the import

## After import

Once the import is complete:

- Verify that all equipment appears correctly in your inventory
- Confirm that employee assignments are correct
- Complete any missing information if necessary

Imported equipment can be modified or updated later from the Primo cockpit.
