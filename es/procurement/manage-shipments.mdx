Este artículo explica cómo crear un envío y configurar la etiqueta de envío asociada. Puede crear un envío durante el flujo de salida o directamente desde la cabina. Los pasos de configuración son los mismos en ambos casos.

Hay dos puntos de entrada:

- Durante el flujo de baja (recomendado cuando se baja un empleado)
- Desde Pedidos y Envíos → Envíos → Nuevo (creación manual)

## Crear un envío

### Del flujo de salida

Utilice este método cuando active una baja. La dirección del empleado se completa automáticamente.

1. Vaya a la página **Desvinculación** del empleado.
2. Vaya al paso **GESTIONAR EQUIPO**.
3. Habilite la opción **Crear una etiqueta de envío**.

Se le redirigirá a la página de creación de envío.

A partir de este punto, siga los pasos de configuración de envío estándar que se describen a continuación.

### Desde la cabina del Primo

Utilice este método para organizar un envío sin pasar por el flujo de salida.

1. Vaya a **Pedidos y Envíos → Envíos → Nuevo**.
2. Seleccione el dispositivo y/o accesorios a enviar.
3. Haga clic en **Siguiente**.

Luego podrá configurar el envío como se describe a continuación.

## Configurar los detalles del envío

Los siguientes pasos se aplican a ambos puntos de entrada.

1. Seleccione o confirme la **dirección del remitente**.
2. Seleccione la **dirección del destinatario**.
3. En **NOTIFICACIONES DE ENVÍO**, agregue las direcciones de correo electrónico que deben recibir la etiqueta de envío (por ejemplo: el empleado, un administrador de recursos humanos o una bandeja de entrada compartida).
4. Seleccione el **perfil de facturación** para definir a quién se le facturará el envío.
5. Opcional: agregue una compra **caja de devolución**. La caja se enviará al remitente para que pueda empaquetar el dispositivo y devolverlo utilizando la etiqueta proporcionada.
6. Haga clic en **Validar** para confirmar y generar la etiqueta de envío.

## Accede a la etiqueta de envío

- La etiqueta de envío se envía por correo electrónico en un plazo de 24 horas a la(s) dirección(es) ingresada(s) en **NOTIFICACIONES DE ENVÍO**.
- La etiqueta también está disponible en Primo Cockpit en **Pedidos y envíos → Envíos → Etiqueta**.

## Proceso de devolución del empleado

### Recibe la etiqueta de envío

El empleado (o cualquier administrador especificado en **NOTIFICACIONES DE ENVÍO**) recibe la etiqueta de envío por correo electrónico dentro de las 24 horas.

### Empaquetar el dispositivo

- Si se compró una caja de devolución: La caja se envía al empleado. Colocan el dispositivo en el interior, aplican la etiqueta de envío y entregan el paquete al transportista.
- Si no se compró ninguna caja: el empleado debe empaquetar el dispositivo de forma segura y aplicar la etiqueta de envío recibida.

### Seguimiento del envío

- La información de seguimiento está disponible a través del correo electrónico de envío.
- El seguimiento también está disponible en Primo Cockpit en **Pedidos y envíos → Envíos**.
