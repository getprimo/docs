Primo le permite automatizar todo el proceso de incorporación para que su equipo no tenga que realizar tareas manualmente cada vez que alguien se une. Una vez configuradas, las incorporaciones se activan automáticamente según las fechas de inicio del contrato y pueden incluir la asignación de dispositivos, la inscripción de MDM y el aprovisionamiento de aplicaciones SaaS.

## Cómo funciona la automatización de la incorporación

La automatización está impulsada por los **datos de sus empleados**. Cuando se detecta un nuevo empleado, ya sea desde su sistema de recursos humanos o agregado manualmente, el sistema puede automáticamente:

- Crear su perfil en la plataforma.
- Activar un flujo de trabajo de incorporación en su fecha de inicio
- Asignar y enviar equipos
- Enviar una invitación de inscripción a MDM
- Proporcione acceso a las aplicaciones que necesitan según su función o equipo

Cuanto más conectada esté su cuenta (HR System, MDM, SaaS), más parte de este proceso podrá ejecutarse sin intervención manual.

## Paso 1: conecte su sistema de recursos humanos

La base de la automatización de la incorporación es la conexión con su sistema de recursos humanos. Cuando se conecta un sistema de recursos humanos, las próximas llegadas se detectan automáticamente en función de las fechas de inicio del contrato y los perfiles de los empleados se crean con anticipación.

Para conectar su sistema de recursos humanos:

1. Vaya a **Configuración > Sincronización de empleados**
2. Seleccione su proveedor de sistemas de recursos humanos (Personio, Lucca, BambooHR y más)
3. Siga los pasos de autenticación
4. Configurar filtros de sincronización y asignaciones de campos
5. Iniciar la primera sincronización.

Una vez conectado, su directorio de empleados se sincroniza cada hora, sin necesidad de realizar ninguna acción manual.

<Note>
Si no utiliza un sistema de recursos humanos, aún puede activar flujos de incorporación agregando empleados manualmente y estableciendo una fecha de inicio. Consulte [Importación mediante CSV](/es/employees/import-via-csv).
</Note>

## Paso 2: asignar equipo

Cuando aparece un nuevo empleado con una fecha de inicio próxima, puedes asignarle equipos directamente desde su perfil. El sistema se encargará del resto:

- Si el dispositivo está **en stock**, se puede preparar y enviar a la dirección del empleado.
- Si solicita un **dispositivo nuevo**, puede configurarlo para Zero Touch Deployment: la máquina llega preinscripta y lista para usar desde el primer momento.

La asignación de equipos se puede realizar tan pronto como se crea el perfil del empleado, mucho antes de su fecha de inicio.

## Paso 3: Automatizar la inscripción de MDM

Una vez que se asigna el dispositivo de un empleado, puede enviar una invitación de inscripción a MDM automáticamente. El sistema:

- Enviar el correo electrónico de inscripción al empleado
- Recordarles automáticamente cada 7 días hasta que se instale el agente MDM.
- Cree y asigne el dispositivo una vez que se complete la inscripción; no es necesario ingresar manualmente

Para enviar invitaciones MDM, vaya a **Empleados > Invitaciones MDM**.

<Tip>
Utilice Zero Touch Deployment si solicita dispositivos mediante adquisiciones. El dispositivo llega con el agente MDM preinstalado: su empleado simplemente lo enciende.
</Tip>

## Paso 4: Proporcionar acceso a la aplicación SaaS

Con SaaS Management, puede otorgar acceso automáticamente a herramientas comerciales según la función, el equipo o la entidad legal de cada empleado, en el momento en que se incorporan.

Para configurar esto:

1. Vaya a **Identidad y acceso > Catálogo de aplicaciones** y conecte sus aplicaciones
2. Definir reglas de aprovisionamiento por rol o equipo
3. Habilitar el aprovisionamiento automático al incorporarse

Cuando se activa la incorporación de un nuevo empleado, el acceso a las aplicaciones adecuadas se proporciona de inmediato, sin tickets ni pasos manuales.

El acceso también se revoca automáticamente cuando se da de baja a un empleado, lo que mantiene limpias sus licencias de SaaS.

## Cómo se ve una incorporación totalmente automatizada

| cuando | ¿Qué sucede automáticamente? |
|---|---|
| **Empleado agregado al sistema de recursos humanos** | Perfil creado, incorporación iniciada |
| **Equipo asignado** | Dispositivo enviado o preparado para Zero Touch |
| **Se acerca la fecha de inicio** | Invitación de inscripción MDM enviada |
| **El empleado enciende el dispositivo** | Agente MDM instalado, políticas aplicadas, aplicaciones enviadas |
| **Incorporación activada** | Aplicaciones SaaS aprovisionadas según el rol y el equipo |

## Requisitos

Para automatizar todo el flujo de incorporación, necesita:

- Un sistema de recursos humanos conectado (o empleados agregados manualmente con una fecha de inicio)
- MDM activado
- Aplicaciones SaaS conectadas con reglas de aprovisionamiento configuradas

Cada paso es independiente: puede automatizar tanto o tan poco como se ajuste a su configuración.
