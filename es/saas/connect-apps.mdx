La solución de gestión SaaS permite a su organización gestionar el acceso a las aplicaciones empresariales de forma segura y centralizada. Al conectar sus aplicaciones, se asegura de que los usuarios se aprovisionen automáticamente, la autenticación esté unificada y los derechos de acceso se apliquen de manera consistente en todas sus herramientas.

Este artículo explica los pasos necesarios para conectar una aplicación.

## Mapeo automatizado de SaaS

Si está utilizando Google Workspace, puede conectarlo en Identidades y Acceso. Mapeará automáticamente todas las aplicaciones para las que se utiliza su Google Workspace como conector SSO.

![Captura de pantalla: [próximamente] Mapeo Saas automatizado](/images/notion-import/connect-your-apps__img-086__block-0005__27343d9f.png)


Una vez conectado, puedes ver todas las identidades relacionadas con cada aplicación y definir un propietario específico.

![Captura de pantalla: [próximamente] Mapeo Saas automatizado](/images/notion-import/connect-your-apps__img-087__block-0007__27343d9f.png)


## Accede al catálogo de aplicaciones

1. Inicie sesión en la cabina Primo.
2. Desde el menú de navegación, vaya a **SaaS**.
3. Seleccione **Identidades** y **Agregar un SaaS**

El [Catálogo de aplicaciones](https://app.getprimo.com/saas-applications/add-app) contiene una lista de aplicaciones que se pueden integrar directamente.

## Seleccionar y configurar la aplicación

![Captura de pantalla: Accede al catálogo de aplicaciones](/images/notion-import/connect-your-apps__img-088__block-0013__25443d9f.png)


1. Localiza la aplicación que deseas conectar.
2. Haga clic en la aplicación para abrir su página de configuración.
3. Siga las instrucciones paso a paso proporcionadas.

Los pasos de configuración varían según el tipo de aplicación:

- **Inicio de sesión único (SSO)**: es posible que deba proporcionar metadatos o certificados para establecer confianza con la aplicación.
- **Aprovisionamiento SCIM**: Es posible que deba ingresar las credenciales API para permitir la creación y actualización automática de cuentas de usuario.
- **Integraciones basadas en API**: algunas aplicaciones requieren que generes e ingreses claves API.

Se proporciona guía en pantalla para que cada aplicación complete la configuración.

![Captura de pantalla: Seleccionar y configurar la aplicación](/images/notion-import/connect-your-apps__img-089__block-0024__25443d9f.png)


## Conclusión

Puede hacer clic en la pestaña Registros y verificar que la sincronización esté activa.

Si cree que su plan es insuficiente para configurar su aplicación, puede utilizar la conexión basada en IA.
