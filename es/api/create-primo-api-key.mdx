---
title: "Crear claves API para Primo"
description: "Genere una clave Primo API para autenticar solicitudes en Primo REST API."
---

Las claves Primo API se utilizan para autenticar solicitudes en [Primo REST API](https://api.getprimo.com/openapi.json). Hay dos tipos de claves API, cada una con un alcance diferente.

## Tipos de claves API

### Clave de empresa API

Una **clave API de empresa** tiene como alcance una única empresa. Brinda acceso a todos los puntos finales que operan con recursos a nivel de empresa: dispositivos, empleados, accesorios, tickets, SaaS y más.

### Clave API del espacio de trabajo

Una **clave API del espacio de trabajo** tiene como alcance todo su espacio de trabajo. Brinda acceso a puntos finales a nivel del espacio de trabajo, como enumerar y ver las empresas dentro de su espacio de trabajo.

<Note>
La mayoría de los terminales API requieren una **Clave API de la empresa**. Solo los puntos finales específicos del espacio de trabajo (bajo `/workspace/`) requieren una **clave API del espacio de trabajo**. Verifique la referencia API para la insignia en cada terminal.
</Note>

## Generar una clave de empresa API

1. Vaya a **Configuración** > **API**.
2. Haga clic en **Crear clave API**.
3. Asigne un nombre a la clave (por ejemplo, "Mi integración").
4. Haga clic en **Crear**.
5. Copie la clave inmediatamente; no se volverá a mostrar.

<Warning>
Guarde su clave API de forma segura. Cualquiera que tenga esta clave puede acceder a sus datos de Primo a través de API.
</Warning>

## Utilice la clave API

Pase la clave en el encabezado `Authorization` de sus solicitudes:

```bash
curl https://api.getprimo.com/v1/devices \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Revocar una clave

Para revocar una clave, vaya a **Configuración** > **API**, busque la clave y haga clic en **Eliminar**.
También puede eliminar una clave API revocada.
