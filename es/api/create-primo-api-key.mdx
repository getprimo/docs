---
title: "Crear claves API para Primo"
description: "Genere una clave Primo API para autenticar solicitudes en Primo REST API."
---

Las claves Primo API se utilizan para autenticar solicitudes en [Primo REST API](https://api.getprimo.com/openapi.json). Existen dos tipos de claves API, cada una con un alcance diferente.

## Tipos de claves API

### Clave API de Empresa

Una **clave API de Empresa** tiene como alcance una sola empresa. Da acceso a todos los endpoints que operan sobre recursos a nivel de empresa — dispositivos, empleados, accesorios, tickets, SaaS y más.

### Clave API de Workspace

Una **clave API de Workspace** tiene como alcance todo su espacio de trabajo. Da acceso a los endpoints a nivel de workspace, como listar y consultar las empresas dentro de su workspace.

<Note>
La mayoría de los endpoints API requieren una **clave API de Empresa**. Solo los endpoints específicos del workspace (bajo `/workspace/`) requieren una **clave API de Workspace**. Consulte la referencia de la API para ver el badge en cada endpoint.
</Note>

## Generar una clave API de Empresa

1. Vaya a **Configuración** > **API**.
2. Haga clic en **Crear clave API**.
3. Asigne un nombre a la clave (por ejemplo, "Mi integración").
4. Haga clic en **Crear**.
5. Copie la clave inmediatamente; no se volverá a mostrar.

<Warning>
Guarde su clave API de forma segura. Cualquiera que tenga esta clave puede acceder a sus datos de Primo a través de la API.
</Warning>

## Utilice la clave API

Pase la clave en el encabezado `Authorization` de sus solicitudes:

```bash
curl https://api.getprimo.com/v1/devices \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Revocar una clave

Para revocar una clave, vaya a **Configuración** > **API**, busque la clave y haga clic en **Eliminar**.
También puede eliminar una clave API revocada.
