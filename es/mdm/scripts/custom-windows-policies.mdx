Amplíe sus perfiles Windows MDM con políticas CSP (proveedor de servicios de configuración) personalizadas utilizando XML sin formato. Utilícelo cuando los controles integrados de Primo no cubran una configuración específica de Windows que necesite implementar, como la personalización del escritorio, el modo quiosco u otras configuraciones empresariales.

## En Windows

<Note>
Al escribir un XML para aplicar una política CSP a un dispositivo, omita las primeras (y últimas) líneas del archivo XML y comience (y finalice) con `<Replace>@@I18N_SEG_10@@<Add>@@I18N_SEG_11@@</Replace>@@I18N_SEG_12@@</Add>`).
</Note>

### Permitir personalización CSP en Windows Pro

```xml
<Replace>
  <!-- Allows Personalization CSPs to be deployed on non-Enterprise versions of Windows.
       Setting this value to true triggers the action to configure a device as education environment.
       Thorough testing is crucial to ensure it does not disrupt workflows or create compatibility issues. -->
  <Item>
    <Meta>
      <Format xmlns="syncml:metinf">bool</Format>
    </Meta>
    <Target>
      <LocURI>./Vendor/MSFT/SharedPC/SetEduPolicies</LocURI>
    </Target>
    <Data>true</Data>
  </Item>
</Replace>
```

### Establecer imagen de fondo y pantalla de bloqueo

<Note>
El CSP de personalización es compatible con las ediciones Windows Enterprise y Education. Funciona en Windows Professional solo cuando está configurado SetEduPolicies en SharedPC CSP. Consulte "Permitir CSP de personalización en Windows Pro".
</Note>

```xml
<Replace>
  <CmdID>1</CmdID>
  <Item>
    <Target>
      <LocURI>
        ./Vendor/MSFT/Personalization/LockScreenImageUrl
      </LocURI>
    </Target>
    <Meta>
      <Format xmlns="syncml:metinf">chr</Format>
      <Type>text/plain</Type>
    </Meta>
    <Data> … your URL here … </Data>
  </Item>
</Replace>
<Replace>
  <CmdID>2</CmdID>
  <Item>
    <Target>
      <LocURI>
        ./Vendor/MSFT/Personalization/DesktopImageUrl
      </LocURI>
    </Target>
    <Meta>
      <Format xmlns="syncml:metinf">chr</Format>
      <Type>text/plain</Type>
    </Meta>
    <Data> … your URL here … </Data>
  </Item>
</Replace>
```
