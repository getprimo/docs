In diesem Artikel wird erläutert, wie Sie eine Sendung erstellen und das zugehörige Versandetikett konfigurieren. Sie können eine Sendung während des Offboarding-Prozesses oder direkt im Cockpit erstellen. Die Konfigurationsschritte sind in beiden Fällen gleich.

Es gibt zwei Einstiegspunkte:

- Während des Offboarding-Ablaufs (empfohlen beim Offboarding eines Mitarbeiters)
- Von Bestellungen & Sendungen → Sendungen → Neu (manuelle Erstellung)

## Erstellen Sie eine Sendung

### Vom Offboarding-Flow

Verwenden Sie diese Methode, wenn Sie ein Offboarding auslösen. Die Adresse des Mitarbeiters wird automatisch vorausgefüllt.

1. Gehen Sie zur **Offboarding**-Seite des Mitarbeiters.
2. Navigieren Sie zum Schritt **GERÄTE VERWALTEN**.
3. Aktivieren Sie den Schalter **Versandetikett erstellen**.

Sie werden zur Seite zur Sendungserstellung weitergeleitet.

Befolgen Sie ab diesem Punkt die unten beschriebenen Standardschritte zur Versandkonfiguration.

### Aus dem Primo-Cockpit

Mit dieser Methode können Sie eine Sendung organisieren, ohne den Offboarding-Prozess durchlaufen zu müssen.

1. Gehen Sie zu **Bestellungen & Sendungen → Sendungen → Neu**.
2. Wählen Sie das zu versendende Gerät und/oder Zubehör aus.
3. Klicken Sie auf **Weiter**.

Anschließend können Sie den Versand wie unten beschrieben konfigurieren.

## Konfigurieren Sie die Versanddetails

Die folgenden Schritte gelten für beide Einstiegspunkte.

1. Wählen oder bestätigen Sie die **Absenderadresse**.
2. Wählen Sie die **Empfängeradresse** aus.
3. Fügen Sie unter **VERSANDBENACHRICHTIGUNGEN** die E-Mail-Adresse(n) hinzu, die das Versandetikett erhalten sollen (z. B. der Mitarbeiter, ein HR-Administrator oder ein gemeinsamer Posteingang).
4. Wählen Sie das **Abrechnungsprofil** aus, um festzulegen, wem die Sendung in Rechnung gestellt wird.
5. Optional: Fügen Sie beim Kauf eine **Rückgabebox** hinzu. Der Karton wird an den Absender geschickt, damit dieser das Gerät verpacken und mit dem bereitgestellten Etikett zurücksenden kann.
6. Klicken Sie auf **Bestätigen**, um den Versandaufkleber zu bestätigen und zu erstellen.

## Greifen Sie auf das Versandetikett zu

- Das Versandetikett wird innerhalb von 24 Stunden per E-Mail an die unter **VERSANDBENACHRICHTIGUNGEN** angegebene(n) Adresse(n) gesendet.
- Das Label ist auch im Primo Cockpit unter **Bestellungen & Sendungen → Sendungen → Label** verfügbar.

## Rückkehrprozess für den Mitarbeiter

### Erhalten Sie das Versandetikett

Der Mitarbeiter (oder ein in **VERSANDBENACHRICHTIGUNGEN** angegebener Administrator) erhält das Versandetikett innerhalb von 24 Stunden per E-Mail.

### Verpacken Sie das Gerät

- Wenn eine Retourenbox gekauft wurde: Die Box wird an den Mitarbeiter verschickt. Sie legen das Gerät hinein, bringen das Versandetikett an und übergeben das Paket dem Spediteur.
- Wenn kein Karton gekauft wurde: Der Mitarbeiter muss das Gerät sicher verpacken und das erhaltene Versandetikett anbringen.

### Verfolgen Sie die Sendung

- Informationen zur Sendungsverfolgung finden Sie in der Versand-E-Mail.
- Die Sendungsverfolgung ist auch im Primo Cockpit unter **Bestellungen & Sendungen → Sendungen** verfügbar.
