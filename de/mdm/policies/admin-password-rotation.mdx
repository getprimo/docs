Rotieren Sie automatisch die Passwörter der lokalen Administratorkonten in Ihrer Flotte und speichern Sie die neuen Anmeldeinformationen sicher in Primo. Dadurch wird das Risiko der Offenlegung von Anmeldeinformationen durch gemeinsame oder veraltete Passwörter verringert, ohne dass ein manueller Eingriff Ihres IT-Teams erforderlich ist.

## Betriebssystemunterstützung

| macOS | Windows | Linux | iOS / iPadOS | Android |
|-------|---------|-------|--------------|---------|
| ✅ | ✅ | | | |

## Wie es funktioniert

Wenn aktiviert, generiert Primo im konfigurierten Rotationsintervall ein neues zufälliges Kennwort für das angegebene lokale Administratorkonto. Das neue Passwort wird sofort im Gerätepanel gespeichert, so dass es für IT-Administratoren jederzeit zugänglich bleibt.

Die Passwortrotation erfolgt stillschweigend – es ist keine Benutzerinteraktion erforderlich und Endbenutzer werden nicht benachrichtigt.

## Konfigurieren Sie die Steuerung

1. Gehen Sie in Primo zu **MDM > Steuerelemente > Neues Steuerelement hinzufügen**.
2. Wählen Sie die Registerkarte Ihres Zielbetriebssystems und wählen Sie **Rotation des Administratorkennworts**.
3. Aktivieren Sie die Steuerung.
4. Legen Sie das **Rotationsintervall** fest (z. B. alle 30, 60 oder 90 Tage).
5. Geben Sie den **Benutzernamen** des zu rotierenden lokalen Administratorkontos ein.
6. Speichern Sie die Steuerung und wenden Sie sie auf die entsprechende Gerätegruppe an.

## Rufen Sie ein rotiertes Passwort ab

Nach einer Rotation ist das aktuelle Passwort im Gerätepanel verfügbar:

1. Gehen Sie zu **Geräte > Geräte** und öffnen Sie das Zielgerät.
2. Navigieren Sie zur Registerkarte **Benutzer**.
3. Klicken Sie auf das Augensymbol neben dem Administratorkonto, um das aktuelle Passwort anzuzeigen.

## Fehlerbehebung

**Das Passwort wurde nicht termingerecht gewechselt**
- Bestätigen Sie, dass das Gerät zum Zeitpunkt der geplanten Rotation online und bei MDM registriert war.
- Überprüfen Sie, ob der Benutzername des Administratorkontos genau übereinstimmt (Groß- und Kleinschreibung bei macOS beachten).
- Stellen Sie sicher, dass die Steuerung noch aktiv und der richtigen Gerätegruppe zugewiesen ist.

**Das abgerufene Passwort funktioniert nicht**
- Möglicherweise ist eine Rotation im Gange. Warten Sie einige Minuten und rufen Sie das Passwort erneut ab.
- Stellen Sie sicher, dass Sie sich das Passwort für das richtige Gerät ansehen – jedes Gerät verfügt über seine eigenen, eindeutigen Anmeldeinformationen.
