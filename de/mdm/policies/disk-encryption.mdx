Schützen Sie Daten auf verlorenen oder gestohlenen Geräten, indem Sie die vollständige Festplattenverschlüsselung in Ihrer gesamten Flotte durchsetzen. Primo aktiviert automatisch die Verschlüsselung während der MDM-Registrierung und hinterlegt Wiederherstellungsschlüssel, sodass IT-Administratoren jederzeit wieder Zugriff erhalten können, ohne sich auf die Anmeldeinformationen des Mitarbeiters verlassen zu müssen.

<Note>
  Wiederherstellungsschlüssel werden nur gespeichert, wenn die Verschlüsselung in Ihren MDM-Sicherheitsrichtlinien > Registerkarte **Profile** aktiviert ist.
</Note>

## Betriebssystemunterstützung

| macOS | Windows | Linux | iOS / iPadOS | Android |
|-------|---------|-------|--------------|---------|
| ✅ (FileVault) | ✅ (BitLocker) | | | |

## Verschlüsselungsfluss

1. Registrieren Sie das Gerät bei MDM
2. Warten Sie 3 bis 4 Stunden, bis der Helfer den Wiederherstellungsschlüssel installiert und verwaltet
3. Starten Sie das Gerät neu (nur für macOS erforderlich)
4. Warten Sie weitere 3 bis 4 Stunden: Der Wiederherstellungsschlüssel wird gespeichert

Wiederherstellungsschlüssel werden pro Gerät auf der Registerkarte **Fleet > Geräte** gespeichert.

Die Verschlüsselungsdauer hängt von der Festplattengröße und der Leistung des Geräts ab.

## Auf macOS

- Die Verschlüsselung wird von **FileVault** verwaltet.
- FileVault wird während der MDM-Installation automatisch aktiviert.
- Die Verschlüsselung wird nach dem nächsten Neustart wirksam.
- Der Wiederherstellungsschlüssel wird zu diesem Zeitpunkt gespeichert.

## Unter Windows

- Die Verschlüsselung wird von **BitLocker** verwaltet.
- Es wird nach der Installation von MDM automatisch aktiviert.
- Es ist kein Neustart erforderlich.
- Der Schlüssel wird gespeichert, sobald die Verschlüsselung aktiviert wird.

## Unter Linux

Primo unterstützt keine automatische Verschlüsselung für Linux-Geräte. Die Aktivierung der Festplattenverschlüsselung unter Linux erfordert normalerweise einen vollständigen Festplatten-Reset und eine Neuinstallation des Betriebssystems, die vor der Registrierung des Geräts manuell durchgeführt werden müssen.
