<Note>
  **Bereitstellungstyp: Automatisch**

  ThreatDown wird **automatisch** über MDM bereitgestellt. Sobald Sie ThreatDown über Primo erwerben und eine MDM-Steuerung erstellen, wird der Agent ohne manuelle Schritte auf alle Zielgeräte übertragen. Es ist keine Benutzerinteraktion erforderlich.
</Note>

In diesem Artikel wird erläutert, wie Sie ThreatDown auf allen oder einem Teil Ihrer Geräteflotte bereitstellen.

## Voraussetzungen

- Habe Primo MDM
- Erwerben Sie eine ThreatDown-Lizenz über Primo

<Note>
  Die automatische ThreatDown-Bereitstellung ist nur für Lizenzen verfügbar, die über Primo erworben wurden. Kontaktieren Sie uns, um loszulegen.
</Note>

## Stellen Sie ThreatDown für Ihre Flotte bereit

Die Bereitstellung von ThreatDown auf den Geräten Ihrer Flotte erfolgt über Ihre MDM-Konfigurationen.

1. Gehen Sie in Primo zu **MDM \> Steuerelemente \> Neues Steuerelement hinzufügen**
2. Wählen Sie die Plattform (verfügbare Plattformen)
3. Aktivieren Sie die Karte **EDR – ThreatDown**
4. Aktualisieren Sie die Ausrichtung, um sie auf die gewünschten Geräte auszurichten
5. Bestätigen

Der ThreatDown-Agent wird automatisch auf allen in der Steuerung enthaltenen Geräten installiert.

## Verfügbare Plattformen

ThreatDown in Primo ist derzeit für die folgenden Plattformen verfügbar:

- **macOS** (arm64) – M1-Chips und mehr
- **Windows** (x64) – Unterstützt Windows Pro

Wenn die Plattform oben nicht aufgeführt ist, können Sie ThreatDown dennoch manuell installieren, indem Sie das Installationsprogramm von der ThreatDown-Konsole herunterladen und es manuell mit Fleet installieren.
