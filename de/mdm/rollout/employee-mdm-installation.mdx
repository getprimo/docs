Dieser Artikel führt Sie Schritt für Schritt durch die Installation von Primo auf Ihrem Gerät. Diese Installation ist für das Flottenmanagement Ihres Unternehmens von entscheidender Bedeutung.

**Bevor wir beginnen, etwas Kontext:**

- **Primo** ist das Tool, mit dem Ihr Unternehmen seine Geräte verfolgt, verwaltet und schützt.
- **FleetDM** ist die von Primo bereitgestellte Mobile Device Management (MDM)-Lösung.

Der Vorgang ist einfach und schnell: Er dauert **weniger als 2 Minuten**.

<Warning>
  **Wichtig**: Sie müssen über **Administratorrechte** auf dem Gerät verfügen, um die Installation abzuschließen. Wenn Sie diese nicht haben, wenden Sie sich an Ihren Administrator, bevor Sie fortfahren.
</Warning>

## Schritt 1: Greifen Sie auf den Registrierungsbereich Primo zu

Sie können auf zwei verschiedene Arten auf den Registrierungsbereich Primo zugreifen.

### Option 1: Greifen Sie über die Einladungs-E-Mail auf die Registrierung zu

Wenn Sie eine Registrierungs-E-Mail mit dem Titel **„Einladung zum Beitritt zu \<Firmenname\> auf Primo“** erhalten haben:

1. Öffnen Sie die Anmelde-E-Mail.
2. Klicken Sie auf **Meinen Computer bei Primo registrieren**.
3. Sie werden zur Primo-Registrierungswebseite weitergeleitet.
    ![Screenshot: Option 1: Zugriff auf die Registrierung über die Einladungs-E-Mail](/images/notion-import/employee-mdm-installation-guide__img-001__block-0017__1b243d9f.png)


### Option 2: Greifen Sie über einen direkten Link auf die Anmeldung zu

Wenn Ihr Administrator einen Registrierungslink mit Ihnen geteilt hat: Öffnen Sie den Link und Sie werden direkt zur Primo-Registrierungswebseite weitergeleitet.

## Schritt 2: Authentifizieren

Sobald Sie auf der Primo-Registrierungsseite sind:

1. Geben Sie Ihre berufliche E-Mail-Adresse ein (oder Ihre persönliche E-Mail-Adresse, wenn Sie neu im Unternehmen sind).
2. Geben Sie den **6-stelligen Bestätigungscode** ein, der an Ihre E-Mail-Adresse gesendet wurde.

## Schritt 3: Starten Sie die Installation

![Screenshot: Schritt 3: Installation starten](/images/notion-import/employee-mdm-installation-guide__img-003__block-0040__2da43d9f.png)


Nach der Anmeldung werden Sie von Primo aufgefordert, MDM auf Ihrem Gerät zu installieren.

1. Wählen Sie eine der folgenden Optionen:
   - Wenn Ihnen bereits ein Gerät zugewiesen ist, wählen Sie es aus und klicken Sie auf **Registrieren**.
   - Wenn Ihnen kein Gerät zugewiesen ist, klicken Sie auf **MDM auf einem anderen Gerät bereitstellen**.
2. Wählen Sie das Betriebssystem Ihres Geräts (Windows, MacOS, Linux, iOS, Android) und laden Sie das Installationsprogramm herunter.
3. Befolgen Sie die Installationsanweisungen für Ihr Betriebssystem.

### Installation auf macOS

1. Installieren Sie den Agenten auf Ihrem Gerät
   1. Suchen Sie die auf Ihr Gerät heruntergeladene Datei **primo-enrollment.mobileconfig**
   2. Doppelklicken Sie auf die Datei. Eine Systemmeldung bestätigt, dass das Profil heruntergeladen wurde und nun installiert werden muss.
       ![Screenshot: Schritt 3: Installation starten](/images/notion-import/employee-mdm-installation-guide__img-002__block-0037__2da43d9f.png)

1. Installieren Sie das Profil auf Ihrem Gerät
   1. Öffnen Sie **Systemeinstellungen** und gehen Sie zu **Allgemein \>Geräteverwaltung** (Sie können auch das Suchfeld in den Systemeinstellungen verwenden und nach Geräteverwaltung suchen.)
   2. Wählen Sie das Profil in der Liste aus und doppelklicken Sie auf „**\<Ihr Unternehmen\> Anmeldung**“.
       ![Screenshot: Schritt 3: Installation starten](/images/notion-import/employee-mdm-installation-guide__img-004__block-0043__1b243d9f.png)

   3. Klicken Sie unten links im Fenster auf **Anmelden**.
       ![Screenshot: Schritt 3: Installation starten](/images/notion-import/employee-mdm-installation-guide__img-005__block-0046__1bc43d9f.png)

   4. Geben Sie Ihr **Administratorkennwort** ein, um die Installation zu bestätigen.

<details>
  <summary>Klicken Sie hier, falls während der Agent-Installation ein Fehler auftritt</summary>

  Wenn Sie eine Fehlermeldung sehen, dass primo-enrollment.pkg blockiert ist:

  ![Screenshot: Schritt 3: Installation starten](/images/notion-import/employee-mdm-installation-guide__img-006__block-0050__2b843d9f.png)

  Sie können die Blockierung unter „Einstellungen“ > „Datenschutz und Sicherheit“ > „Sicherheit“ aufheben

  ![Screenshot: Schritt 3: Installation starten](/images/notion-import/employee-mdm-installation-guide__img-007__block-0052__2b843d9f.png)

</details>

Die Anmeldung ist abgeschlossen. Sie erhalten eine Bestätigungs-E-Mail.

### Installation auf Windows

1. Doppelklicken Sie auf die heruntergeladene Datei „**primo-enrollment.msi**“.
2. Es öffnet sich ein Windows-Sicherheitsfenster. Fahren Sie fort, indem Sie auf **„Ja“** klicken.
    ![Screenshot: Schritt 3: Installation starten](/images/notion-import/employee-mdm-installation-guide__img-008__block-0058__1b243d9f.png)

3. Die Anmeldung ist abgeschlossen. Sie erhalten eine Bestätigungs-E-Mail.

### Installation auf iOS

1. Gehen Sie zu **Einstellungen** \> **Allgemein** \> **Geräteverwaltung**
2. Wählen Sie das Profil aus und **installieren** Sie MDM Primo
3. **Bestätigen Sie die Installation**, indem Sie bei Aufforderung Ihren Freischaltcode eingeben

### Installation auf Android

1. Befolgen Sie die Registrierungsschritte für Android
2. Wenn Sie zur Eingabe der Anmeldeinformationen aufgefordert werden, geben Sie Ihre Primo-Anmeldeinformationen ein

<Note>
  Bei Primo hat der Schutz Ihrer Daten für uns absolute Priorität. Weitere Informationen finden Sie auf unserer Transparenzseite.
</Note>
