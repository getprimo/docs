In diesem Artikel wird erläutert, wie Sie Ihr Mitarbeiterverzeichnis manuell verwalten, wenn kein HR-System angeschlossen ist.

## Anwendungsfälle

Verwenden Sie diese Methode, wenn:

- Sie nutzen kein HR-System
- Sie führen einen einmaligen Import durch
- Sie möchten externe Mitarbeiter hinzufügen (Freiberufler, Praktikanten usw.)

Beachten Sie, dass durch den Import Ihrer Mitarbeiter die Ankünfte und Abgänge Ihrer Mitarbeiter manuell in Primo eingegeben werden müssen.

## Manueller Mitarbeitersynchronisierungsprozess

Hier sind die Informationen, die Sie in Primo importieren können und was Ihre CSV-Datei enthalten muss, um Mitarbeiter korrekt zu importieren:

- Vorname \*
- Nachname \*
- Professionelle E-Mail \*
- Persönliche E-Mail (optional)
- Telefonnummer (optional)
- Team (optional)
- Berufsbezeichnung (optional)

Unten auf dieser Seite finden Sie eine Vorlage, die Sie herunterladen und ausfüllen können, um den Import Ihrer Mitarbeiter zu erleichtern.

So importieren Sie Mitarbeiter manuell:

1. Gehen Sie zu **Einstellungen \> Mitarbeitersynchronisierung**
2. Klicken Sie auf „CSV oder Mitarbeiter hochladen“
3. Wählen Sie die Datei aus, die Sie importieren möchten
4. Ordnen Sie die Titel von Primo denen in Ihrer Datei zu
5. Überprüfen Sie die Informationen und importieren Sie sie
6. Klicken Sie auf **Speichern**. Der Mitarbeiter wird in Ihrer Mitarbeiterliste angezeigt

## Manuelle Updates

- Sie können jedes Feld jederzeit **bearbeiten**
- Felder werden **niemals überschrieben**, es sei denn, Sie verbinden später ein HR-System

<Warning>
  **Einschränkungen**

  - Keine automatisierten Onboarding-/Offboarding-Abläufe
  - Keine Synchronisierung mit Vertragsdaten
  - Bei Ausscheiden von Mitarbeitern ist eine manuelle Bereinigung erforderlich
</Warning>

## Vorlagendatei

Laden Sie die Vorlagendatei herunter, um den Import Ihrer Mitarbeiter zu erleichtern. Die Vorlage enthält Spalten für alle unterstützten Felder.
