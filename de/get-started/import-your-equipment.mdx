<Note>
Durch die Installation des MDM auf den Geräten Ihrer Mitarbeiter können diese automatisch im Cockpit erstellt werden, ohne dass eine manuelle Aktion oder ein Import Ihrerseits erforderlich ist. Nach der Registrierung erscheint das Gerät in Ihrem Fleet-Tab und wird dem Mitarbeiter zugewiesen.
</Note>

Wenn Sie sich dennoch für den Import Ihrer Ausrüstung entscheiden, sollten Sie die folgenden Informationen beachten.

## Voraussetzungen

Um Geräte in Primo zu importieren, ist lediglich ein Name erforderlich. Um jedoch eine gute Sicht auf Ihre Flotte zu haben, empfehlen wir, so viele Informationen wie möglich zu importieren. Dies hilft Ihnen, genaue Aufzeichnungen zu führen.

## Vorbereitung der Importdatei

Hier ist eine Vorlagendatei, die Ihnen den einfachen Import Ihrer Ausrüstung erleichtert:

| **Kriterien** | **Erforderliches Format** | **Pflichtfeld?** |
|---|---|---|
| Mitarbeiter-E-Mail (falls Gerät zugewiesen) | E-Mail | Nein |
| Gerätetyp | LAPTOP<br/>SMARTPHONE<br/>TABLET | Ja |
| Gerätename | | Ja |
| Betriebssystem | WINDOWS<br/>MACOS<br/>LINUX<br/>ANDROID<br/>IOS<br/>UNBEKANNT | Nein |
| Seriennummer | | Nein |
| Bildschirmgröße | | Nein |
| Tastatursprache | FRANZÖSISCH | Nein |
| Marke | | Nein |
| Jahr | | Nein |
| Notizen | | Nein |

## Importvorgang

So importieren Sie Ihre Ausrüstung in das Primo-Cockpit:

1. Melden Sie sich in Ihrem Primo-Cockpit an
2. Gehen Sie zur Registerkarte Mein Fleet
3. Benutzen Sie die Schaltfläche „Hinzufügen“.
4. Wählen Sie Ihre vorbereitete CSV-Datei aus
5. Überprüfen Sie die Spaltenzuordnung
6. Bestätigen Sie den Import

## Nach dem Import

Sobald der Import abgeschlossen ist:

- Stellen Sie sicher, dass alle Geräte in Ihrem Inventar korrekt angezeigt werden
- Bestätigen Sie, dass die Mitarbeiterzuordnungen korrekt sind
- Ergänzen Sie ggf. fehlende Angaben

Importierte Geräte können später über das Primo-Cockpit geändert oder aktualisiert werden.
